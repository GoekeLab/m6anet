.. _help:

Getting Help
==================

We appreciate your feedback and questions! You can report any error or suggestion related to m6Anet as an `issue on github <https://github.com/GoekeLab/m6anet/issues>`_. If you have questions related to the manuscript, data, or any general comment or suggestion please use the `Discussions <https://github.com/GoekeLab/m6anet/discussions>`_.

Thank you!
