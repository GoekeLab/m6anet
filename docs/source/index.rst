Welcome to m6anet's documentation!
==================================
m6anet is a python tool to detect m6a modifications from Nanopore Direct RNA Sequencing data

To install our m6anet package and its dependencies, run::

        python setup.py install
        pip install -r requirements.txt

See our :ref:`Installation page <installation>` for details.

To detect modification from your direct RNA sequencing smaple, you can follow the instructions in our :ref:`Quickstart page <quickstart>`.
