Installation
=======================

m6Anet requires `Python3 <https://www.python.org>`_ to run.

PyPI installation (recommended)
---------------------------------
::

    pip install m6anet

Installation from our GitHub repository
---------------------------------------
::

    git clone https://github.com/GoekeLab/m6anet.git
    cd m6anet
    python setup.py install
